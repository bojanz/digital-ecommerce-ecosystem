Commerce License

What is it?

It's a framework for selling access to local or remote resources.

Features

A license can be configurable and have fields that are filled-in by the user

Features

A license can be remote/synchronized with a remote service.

Features

Licenses can be time limited, and are automatically expired.

Commerce License (features)

  • Licenses can be time limited, and are automatically expired (by cron) once that time passes.
  • Remote licenses are synchronizable.
  • A checkout pane shows the access details in the complete page during checkout.
Talk about advandcedqueue. The remote service can be contacted each time the status changes.

Commerce License billing (features)

  • Handles prepaid/postpaid products.
  • Handles prorated payments.
  • Handles recurring billing for licenses.
  • Handles usage based billing.
Very complex to build, doesn't have a release yet. Speak about the billing cycle types.
